## What is Linux?

Linux is a free, open-source operating system used by most
servers, cloud platforms and DevOps tools in the world.

When you connect to AWS EC2 or Azure VM
you are connecting to a Linux machine!

---

## Why Linux for DevOps?

| Reason | Details |
|--------|---------|
| Free | No license cost |
| Stable | Runs for years without restarting |
| Secure | Better permission control |
| Industry standard | 90% of cloud servers run Linux |
| DevOps tools | Docker, Kubernetes, Terraform all run on Linux |
